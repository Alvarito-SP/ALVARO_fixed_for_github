# ALVARO_fixed_for_github
Portfolio – Alvaro Quintanilla

This project is a professional personal portfolio designed to showcase projects, skills, experience, and a contact page. It is built using HTML, CSS, Bootstrap, and static assets, and it is fully compatible with GitHub Pages.

Features
Modern and clean design
Fully responsive layout using Bootstrap
Projects section with images and live links
Dedicated resume page
Functional contact form (SB Forms or custom backend)
Easy navigation between Home, Projects, Contact, and Resume
Fully optimized for GitHub Pages deployment
